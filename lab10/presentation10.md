# Цель работы: 
 
Изучить основы программирования в оболочке ОС UNIX/Linux, научиться писать небольшие командные файлы.
 
# Ход работы:

При этом файл должен архивироваться одним из архиваторов на выбор zip, bzip2 или tar.
